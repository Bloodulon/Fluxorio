# Fluxorio

ASCII игра в стиле Factorio с процедурной генерацией мира, конвейерными системами и крафтом.

## Управление

### Движение
| Клавиша | Действие |
|---------|----------|
| WASD / ЦФЫВ | Движение |
| Стрелки | Движение |

### Строительство
| Клавиша | Действие |
|---------|----------|
| 1 | Режим: Конвейер |
| 2 | Режим: Печь |
| 3 | Режим: Инсертер |
| 4 | Режим: Разрушить |
| Space | Построить |
| O | Повернуть конвейер |
| R | Отменить режим |

### Взаимодействие
| Клавиша | Действие |
|---------|----------|
| E / У | Собрать ресурс |
| C / С | Открыть меню крафта |
| Esc | Закрыть меню крафта |

### Крафт
| Клавиша | Рецепт |
|---------|--------|
| 1 | Железная руда → Железная пластина |
| 2 | Медная руда → Медная пластина |
| 3 | Уголь (скоро) |

## Особенности

- Процедурная генерация карты 200x200
- Система конвейеров с движением предметов
- Печи с рецептами крафта
- Инсертеры (анимация состояний)
- Инвентарь игрока
- Меню крафта в реальном времени
- Поддержка русской и английской раскладки клавиатуры

## Сборка

```bash
cargo build --release
```

## Запуск

```bash
cargo run
```

или

```bash
cargo run --release
```
